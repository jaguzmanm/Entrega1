Projects

Automatic Detection of Points

During the robotics course from the IELE department, I worked along with some classmates in an application to automatically detect coordinates of certain points of a live streaming from a camera. This application is particularly useful when programming a robot because it allows the robot to position himself in the environment.

GUI to control a Rover

I was a member in the student group RoboCol, in there I worked with the interface group, the purpose of the group was to make a website that connects with all the information from ROS topics given by the rover hardware, process this information and present it to the user. In the other hand, the communication was bidirectional, in the way that you could also control the rover using the GUI designed for that purpose.
